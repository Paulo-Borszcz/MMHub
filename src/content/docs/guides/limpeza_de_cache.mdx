## Introdução

Limpar o cache do navegador pode resolver muitos problemas de navegação, incluindo erros de carregamento de páginas e problemas de login. Este guia detalha os passos para limpar o cache nos navegadores Google Chrome e Mozilla Firefox.

<Tabs>
  <TabItem label="Chrome" icon="chrome">
    ## Limpeza de Cache no Google Chrome

    **Passos:**
    <Steps>
    1. Abra o navegador Chrome.
    2. Clique no menu de três pontos no canto superior direito.
    3. Selecione "Mais ferramentas" e depois "Limpar dados de navegação".
    4. No menu suspenso, selecione o intervalo de tempo desejado.
    5. Marque a opção "Imagens e arquivos armazenados em cache".
    6. Clique em "Limpar dados".
    </Steps>

  </TabItem>
  <TabItem label="Firefox" icon="firefox">
    ## Limpeza de Cache no Mozilla Firefox

    **Passos:**
    <Steps>
    1. Abra o navegador Firefox.
    2. Clique no menu de três linhas no canto superior direito.
    3. Selecione "Opções".
    4. No painel esquerdo, clique em "Privacidade e Segurança".
    5. Role para baixo até a seção "Cookies e Dados do Site".
    6. Clique em "Limpar dados".
    </Steps>

  </TabItem>
    <Aside type="note">Os passos podem variar um pouco dependendo da versão do navegador.</Aside>
</Tabs>
